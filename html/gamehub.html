Game Hub Info

The Game Hub Application is a sleek and modern web-based platform designed to provide users with easy access to a variety of engaging games. This application features a visually appealing and user-friendly interface, leveraging Bootstrap for responsive design and Font Awesome icons for intuitive navigation.


Features:
Word Guessing Game

A fun and challenging game where players guess the correct word based on given hints.
Connect 4 Game

The classic strategy game where players take turns dropping discs into a grid, aiming to connect four of their pieces in a row.
Memory Game

A cognitive game designed to test and improve memory skills by matching pairs of cards.
Simon Trickster Game

A game that tests memory and pattern recognition by challenging players to repeat increasingly complex sequences of lights and sounds.
2048 Game

A popular puzzle game where players combine numbered tiles to reach the elusive 2048 tile.
Tic Tac Toe

The timeless two-player game where players aim to get three of their marks in a row on a 3x3 grid.


Technologies Used:
  • Bootstrap

    For responsive layout and styling

  • Font Awesome

    For visually appealing icons that enhance user experience.

  • HTML/CSS

    For the structure and design of the application.

  • JavaScript

    For interactivity and game logic.



Navigation:
Users can easily navigate through the game hub by clicking on the respective icons, each representing a different game. The application is designed to be intuitive and responsive, ensuring a seamless experience across various devices.
