Lunch Order App Info

During my time at IT Certification Academy, I developed an application called the Lunch Order App. They requested this project in order to reduce the hassle of clients having to write down their lunch order on paper every time. In reality, this project does not place orders with the delivery services. Users may register, browse the menu, select what they want, and it will be logged for administrators to view. They can then assign someone to go to the shop and have the customers' requests fulfilled.

Technologies Used:
  • ASP.NET Core MVC

    For its flexibility, performance, and robustness.

  • Bootstrap

    For responsive layout, styling amd icons

  • HTML/CSS

    For the structure and design of the application.

  • JavaScript

    For interactivity and API's
